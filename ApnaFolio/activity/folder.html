TreeHouse

Tree data structure is very important specially interms of it's visualizations.

TreeHouse is all about the clean visualization of tree data structure of many forms say Binary tree, BST, Heaps, recursive tree as well.

Most of the times our laziness overrides us and we don't tend to draw the tree, thereby lacking in visualizations specially at interviews.

Also drawing specially in recursion can get messy.

Treehouse maintains clean visuals of the tree and you can play around it.

Draw, travel, play with your tree using tools.

Stores your tree snaps, videos and revise before interviews.

You can teach your peers or students with clean implementation and visuals.

Content creators can record, share, upload your solutions on YouTube or coding discussion platforms. Can use the green screen(chroma) effect for background.

BFS algorithm implementation for tree structure.

ExcelBook

Calculations can be performed on cells with respect to formula by Infix evaluation using Stacks Data Structures.

Two-way binding on all cells with it's own properties mapped and highlighted.

Formula/Data updation in a cell leads to updation on dependent cells in a linked chain fashion throughout.

Cyclic links of formulas detected by cycle detection algorithm using Graphs Data Structures.

Trace your regular chained calculation path from any cell and cyclic calculation path with color tracing feature.

Multiple sheets handling, download sheets, open sheet file, CRUD operations performed.

Local Storage for current sheet's data retrieval.

Responsive UI to enhance user experience across all devices.

JobReady

JobReady scraps(collects) all previously interviewed questions of requested company from GeeksForGeeks with their frequency count (of higher importance)

Automation tasks are carried out using the Node.js library, Puppeteer.
